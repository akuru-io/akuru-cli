# Akuru CLI

How to run the project

### MacOS
```
$ cd akuru-cli
$ python3 -m venv venv
$ source venv/bin/activate
(venv)$ pip install -r requirements.txt
```

### CLI Usage
How to install a font  
```
$ python3 main.py install "Abhaya Libre"
```
How to uninstall a font
```
$ python3 main.py uninstall "Abhaya Libre"
```
